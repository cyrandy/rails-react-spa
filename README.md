# rails-react-spa

## Env Requirement

- Ruby
- Node >= v4.0.0
- Rails v3.2

## Setup

- `bundle install`
- `npm install`

## Start Dev Server

- `npm run build`
- `rails s`
- open a browser tab to `localhost:3000`

## Start Production Server

- `npm run build`
- `rake assets:precompile`
- `rails s -e production`
